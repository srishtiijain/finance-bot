# FinanceBot 
An AI-powered Telegram bot designed to simplify financial concepts for Indian users. Built using Python, the Gemini API, and deployed on Render.

##  Features
* **Real-time Financial Advice:** Get instant answers to queries about SIPs, ELSS, Tax Saving (80C), and more.
* **Powered by GenAI:** Uses Google's Gemini Pro model for natural, context-aware conversations.
* **24/7 Availability:** Hosted in the cloud to provide assistance anytime.

## Tech Stack
* **Language:** Python 3.10+
* **Framework:** `pyTelegramBotAPI`
* **AI Model:** Google Gemini Pro
* **Deployment:** Render (Background Worker)
